Browser Games

Monorepo home for web-playable tools and games.

Grapple and Go

  • Grapple and Go — Host Screen (laptop/TV, shows QR code for players)
  • Grapple and Go — Join as Player (or scan QR from host screen)

Swing as far right as you can using grappling hooks. Each player joins from their phone by scanning the QR code. Each player takes a turn; furthest distance wins.

Boggle

  • Boggle — Dedicated Host Screen (laptop/TV, shows QR code for players)
  • Boggle — Host + Play on Phone (start a game and play too)
  • Boggle — Join as Player (or scan QR from host screen)

Boggle multiplayer runs serverlessly on GitHub Pages via PeerJS WebRTC. Scan the QR code on the host screen to join.

Stones of Five

  • Stones of Five — Dedicated Host Screen (laptop/TV, shows QR code for players)
  • Stones of Five — Host + Play on Phone (start a game and play too)
  • Stones of Five — Join as Player (or scan QR from host screen)

Stones of Five multiplayer runs serverlessly on GitHub Pages via WebRTC. 2–4 players on a 19×19 board. Win by five-in-a-row or five capture pairs.

Worlds Collide — FF6 Randomizer

  • Worlds Collide WASM Prototype

Browser-based prototype of the Worlds Collide Final Fantasy VI randomizer, running the Python logic via WebAssembly.
