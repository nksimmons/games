Pente

  • Pente — Dedicated Host Screen (laptop/TV, shows QR code for players)
  • Pente — Host + Play on Phone (start a game and play too)
  • Pente — Join as Player (or scan QR from host screen)

Pente multiplayer runs serverlessly on GitHub Pages via PeerJS WebRTC. 2–4 players on a 19×19 board. Win by five-in-a-row or five capture pairs.
