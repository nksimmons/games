# games

Monorepo for browser-playable projects hosted on GitHub Pages.

## Apps

- worlds-collide WASM prototype: worldscollide/

## Local Preview

Run any static file server at repo root, for example:

```sh
npx serve .
```

Then open:

- /
- /worldscollide/

## GitHub Pages

This repo includes a `.nojekyll` file so Python package files such as `__init__.py` are served correctly.
